Box Model em CSS

Conceito fundamental que determina o posicionamento e dimenssões dos elementos na página.

Elementos

Elementos Definição
Margin Define as margens dos elementos
Border Define as bordas/contorno dos elementos
Padding Define a distância interna entre o elemento e sua borda
Content Definida para gerar conteúdo em um
elemento, como por exemplo, quanto selecionamos um botão (active),
o conteúdo do botão sera selecionado
